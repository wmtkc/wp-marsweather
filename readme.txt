== Description ==

Created for Insight Designs

Widget displays atmospheric temperature, pressure, and wind data from NASA's InSight lander for the last five sols.
Data fetched from [NASA InSight Mars Weather Service API](https://api.nasa.gov/).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory
2. Activate the plugin through the 'Plugins' admin page in WordPress
3. Add the widget to your public page through the 'Appearance->Widgets' admin page in WordPress. Note that the widget takes up the full screen width on both mobile and desktop.
4. Use the Mars Weather Widget admin page to configure the plugin
